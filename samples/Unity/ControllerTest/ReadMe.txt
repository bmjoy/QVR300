Controller Test
-----------------------------------------------------------

--------------
Installation
--------------
Run ControllerTest\install.bat to Install ControllerTest.apk
Run XimmerseControllerService\install.bat to Install XimmerseControllerService.apk
Install BTConfig[1.0.0-Flip].apk from https://github.com/Ximmerse/SDK_Flip/tree/master/Tools

----------------------------
XimmerseControllerService
----------------------------
This is the service that talks to Ximmerse Controller Daemon and provides the values to the SVR SDK in a format that is required by the SVR SDK.

----------------------------
Ximmerse Flip SDK Daemon
----------------------------
This is provided by Ximmerse and has the service that talks to the bluetooth controller.

-------------------------------
Pairing the controller
-------------------------------
Launch XimmerseControllerService app and select Bind
Press and hold the Back button then press and hold the Start button.
See xImmerse3Dof.PNG for buttons.
